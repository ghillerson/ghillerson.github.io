Start Here!

This page points you to the top-level documentation pages you need to visit to quickly get started with installing and using Splice Machine:

  1. Check our requirements page to make sure that your hardware, cluster (if applicable), and software environment will work with Splice Machine.

  2. You can get started quickly with a cluster sandbox on Amazon Web Services by following the instructions in the Installing Splice Machine on Amazon Web Services topic in our Administrator's Guide.

    Or you can use Splice Machine Installation Guide to install Splice Machine on your cluster or standalone computer, following the step-by-step instructions for your platform.

  3. For access to the full source code for Splice Machine, visit our open source GitHub repository: https://github.com/splicemachine/spliceengine

  4. Review our SQL Reference Manual, especially our Built-in System Procedures section, which features functions with which you might not be familiar.
  5. Use the splice> command line interface to explore the database.
  6. Import data into your Splice Machine database.
  7. Configure any maintenance tasks (such as backups) that you want to run periodically.
  8. Explore connecting your database with Business Intelligence tools and other databases.
  9. Enjoy using Splice Machine!
