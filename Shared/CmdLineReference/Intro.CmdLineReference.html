Command Line (splice>) Reference

This section describes the commands that are available from the splice> command line.

You can also enter SQL statements directly on the splice> command line. You can find information about the SQL statements in the SQL Reference Manual.

This section contains the following topics:

Topic Description
Commands Summary Contains a table summarizing all of the splice> commands you can use, in addition to using SQL on the command line.
Command Line Syntax Describes the syntax recognized by the splice> command line interpreter.
rlWrap Synopsis Provides a summary of common keyboard commands you can use with rlWrap, which is a Unix command line utility, recommended by Splice Machine, that provides history and command line alteration when using the splice> command line.
General Commands General purpose splice> commands.
Show Commands The show commands you can use to display various information about your database.
Connection Commands Commands for establishing, using, and disconnecting database connections.
Transaction Commands Commands for working with transaction savepoints and transaction commits.
Optimization and Statistics Commands Commands for working with Splice Machine statistics to optimize your queries.
