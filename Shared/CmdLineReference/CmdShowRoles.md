[]()Show Roles
==============

The <span class="AppCommand">show roles</span> command a sorted list of all of the roles defined in the database.

Syntax

``` FcnSyntax
SHOW ROLES
```

Examples

``` AppCommand
splice> create role testRole;
0 rows inserted/updated/deleted
splice> show roles;

ROLEID
------------------------------
TESTROLE

1 row selected
```
