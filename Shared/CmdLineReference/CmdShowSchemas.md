[]()Show Schemas
================

The <span class="AppCommand">show schemas</span> command displays all of the schemas in the current connection.

Syntax

``` FcnSyntax
SHOW SCHEMAS 
```

Examples

``` AppCommand
splice> show schemas;
TABLE_SCHEM
------------------------------
NULLID
SPLICE
SQLJ
SYS
SYSCAT
SYSCS_DIAG
SYSCS_UTIL
SYSFUN
SYSIBM
SYSPROC
SYSSTAT
11 rows selected
```
