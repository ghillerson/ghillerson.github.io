SQL Reference Manual
====================

This section contains reference information for Splice Machine SQL. Our implementation includes all of ANSI SQL-99 (SQL3), with added optimizations and features.

Note that this section is modeled on and borrows heavily from the SQL Reference section of the Apache Derby 10.9 documentation, as permitted by the Apache License. This SQL Reference Manual contains the following sections:

| [Data Types](DataTypes/Intro.DataTypes.html)                                           | Describes the data types used in SQL.                                                                       |
| [Statements](Statements/Intro.Statements.html)                                         | Reference pages for our implementation of each SQL statement.                                               |
| [Clauses](Clauses/Intro.Clauses.html)                                                  | Reference pages for our implementation of SQL clauses.                                                      |
| [Expressions](Expressions/Intro.Expressions.html)                                      | Describes the expressions you can use in SQL.                                                               |

| [SQL Built-in Functions](BuiltInFcns/Intro.BuiltInFcns.html)                           | Reference pages for the standard SQL functions featured in our implementation.                              |
