Connecting Your Splice Machine Database

This section contains the following topics to help with connecting your Splice Machine database with other tools and databases:

Section Describes
Using our JDBC Driver Describes how to find, configure, and use the Splice Machine JDBC driver. This topic includes several examples of connecting programmatically to your Splice Machine database with different programming languages.
Using our ODBC Driver Describes how to find, configure, and use the Splice Machine ODBC driver. This topic includes an example of connecting programmatically to your Splice Machine database with the C programming language.
Connecting with Data Tools How to connect your Splice Machine database with specific Business Intelligence tools, including DBeaver, DBVisualizer, Pentaho PDI, Squirrel, and Tableau.
